Laurel's New Media & Tech for Social Change Website

The Three Big Projects

In my New Media class, we had three big projects: The Black Box Investigation project, the Shared Canvas project, and the DNA Anaylsis project. In each project we were placed with a team of students that would help each other throughout the project. Moreover, we developed our 21st Century Skills: Problem Solving and Critical Thinking, Accessing and Analyzing Information, Communication and Collaboration, Creativity and Innovation Initiative, Self-Direction and Accountability, and Citizenship and Responsibility. Above you can find each of the projects in their own tab.

This Website's Creation

To create this website, we used Github. First we had to make our own repositories, then the files. We created seperate files for each of the tabs we wanted. For example, I had a "blackbox.html" file for the Black Box Investigation but a "sharedcanvaslaurel.html" for the Shared Canvas. In addition, I created a "style.css" file to stylize the websites. For the website, to get an A we had lots of requirements including: having at least three complete webpages about course projects, a functioning homepage at index.html with navigation links that help users move between its pages and a bit of information about how we made this website, images, and your own custom-written style.css code.
